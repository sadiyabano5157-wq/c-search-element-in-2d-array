# c-search-element-in-2d-array
A C program to search for a number in a 3×3 two-dimensional array using nested loops and a flag variable.

# Search Element in 2D Array (C)

## Description
This project searches for a user-entered number in a 3×3 two-dimensional array. If the number exists, the program displays a success message; otherwise, it displays that the number was not found.

## Concepts Used
- C Programming
- 2D Arrays
- Nested Loops
- User Input
- if-else Statements
- Flag Variable

## Array
10 20 3
40 50 60
7  80 90

## Example

Input:
50

Output:
Search number found

Input:
25

Output:
Search number not found
